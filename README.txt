Cannon Blast

A mobile application designed for devices that run using the Android operating system.

The goal of the game is to eliminate all of the target pieces before the time runs out.
The quicker the user can destory all the pieces, the higher the score they will recieve.
2 seconds are added to the clock when a target is hit.
3 seconds are removed from the clock when the blocker is hit.

Source code is in the app folder.

Import this whole repo into Android studio to run.
This software will run on an android device or through the Android Emulator.
